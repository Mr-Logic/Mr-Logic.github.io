hexo搭建博客托管在github上基本步骤

1.新建GitHub仓库，仓库名为：username.github.io

2.安装Git、Node.js

3.安装hexo脚手架

npm install -g hexo-cli -g

4.创建博客

hexo init username.github.io

5.更改配置

打开配置文件username.github.io/_config.yml

title: dimsky 的 9 维空间 //你博客的名字 
author: dimsky //你的名字
language: zh-Hans //语言 中文
theme: landscape //默认主题
deploy:
    type: git //使用Git 发布
    repo: git@github.com:username/username.github.io.git // 刚创建的Github仓库地址，需要添加ssh key到github账号
    branch: master

6.新建文章

hexo new article_name

打开文件夹username.github.io/source/_posts，打开刚才的新建的文件，编写即可。

7.本地测试

hexo server

然后打开localhost:4000端口即可浏览

8.安装hexo-deployer-git自动化部署

npm install hexo-deployer-git --save

9.发布到github

hexo clean && hexo g && hexo d

若第8步执行完后执行第9步报如下错误

ERROR Local hexo not found in 本地磁盘
ERROR Try running: 'npm install hexo --save'

则运行如下命令即可

npm install

浏览器打开https://username.github.io即可访问博客

10.每次写完文章后执行命令9即可同步到github

11.若需要更改主题，将主题clone到themes目录下，再修改配置文件中的theme为clone下的文件名，并theme文档修改配置即可

参考链接：
https://www.jianshu.com/p/4eaddcbe4d12
