create-react-app项目中使用less和antd基本配置

1.安装less和less-loader

yarn add less-loader
yarn add less@2.7.3        //less版本大于3.0时会报错，最好是2.7-2.9的版本
                        //之所以需要支持less是因为antd是基于less开发的，脚手架创建的项目本身不支持less

2.暴露webpack配置

yarn eject

3.修改webpack配置以解析less文件

将config文件夹下webpack.config.dev.js中如下代码拷贝一份，放到它的后面

{
    test: /\.css$/,
    use: [
      require.resolve('style-loader'),
      {
        loader: require.resolve('css-loader'),
        options: {
          importLoaders: 1,
        },
      },
      {
        loader: require.resolve('postcss-loader'),
        options: {
          // Necessary for external CSS imports to work
          // https://github.com/facebookincubator/create-react-app/issues/2677
          ident: 'postcss',
          plugins: () => [
            require('postcss-flexbugs-fixes'),
            autoprefixer({
              browsers: [
                '>1%',
                'last 4 versions',
                'Firefox ESR',
                'not ie < 9', // React doesn't support IE8 anyway
              ],
              flexbox: 'no-2009',
            }),
          ],
        },
      }
    ],
  },
}

将上面的代码修改两处地方：最开始的css改为less，最后添加{loader: require.resolve(‘less-loader’)}

{
    test: /\.less$/,
    use: [
      require.resolve('style-loader'),
      {
        loader: require.resolve('css-loader'),
        options: {
          importLoaders: 1,
        },
      },
      {
        loader: require.resolve('postcss-loader'),
        options: {
          // Necessary for external CSS imports to work
          // https://github.com/facebookincubator/create-react-app/issues/2677
          ident: 'postcss',
          plugins: () => [
            require('postcss-flexbugs-fixes'),
            autoprefixer({
              browsers: [
                '>1%',
                'last 4 versions',
                'Firefox ESR',
                'not ie < 9', // React doesn't support IE8 anyway
              ],
              flexbox: 'no-2009',
            }),
          ],
        },
      },
      {
        loader: require.resolve('less-loader')
      }
    ],
  },

####同样webpack.config.prod.js也按照此方法修改即可

4.安装antd

yarn add antd

5.安装babel-plugin-import实现antd按需加载

yarn add babel-plugin-import

6.修改webpack配置以支持antd按需加载

####在webpack.config.dev.js中找到如下代码
{
test: /.(js|jsx|mjs)$/,
include: paths.appSrc,
loader: require.resolve(‘babel-loader’),
options: {

        compact: true,
    },
}

在上面的代码中的options中添加配置，修改后为如下即可：

{
    test: /\.(js|jsx|mjs)$/,
    include: paths.appSrc,
    loader: require.resolve('babel-loader'),
    options: {
        plugins: [
            ['import', [{libraryName: 'antd', style: true}]]
        ],
        compact: true,
    },
}

在webpack.config.prod.js中同样修改如上。至此，即可在react项目中按需加载antd

// 需要注意的是dev.js和prod.js中的修改要保持一致以保证开发版本配置和生产版本配置一致。
