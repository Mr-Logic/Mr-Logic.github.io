React中引入本地图片和背景图片

1.非背景图片引入方式

①-直接在img标签内部src中使用

<img src={require('图片路径')} />

②-import导入

import imgSrc from '图片路径'
<img src={imgSrc} />

2.图片作为背景引入

①-render()方法内定义为对象

const bgGround {
    display: 'inline-block',
    height: '40px',
    width: '40px',
    background: `url(${require("图片路径")})`
}

// 在return中使用
<span style={bgGround}>xxxxx</span>

②-import引入

import imgUrl from '图片路径'

// render()中定义为对象
const bgGround = {
    display: 'inline-block',
    height: '40px',
    width: '40px',
    backgroundImage: 'url(' + imgUrl + ')'
}

//在return中使用
<span style={bgGround}>xxxxx</span>
