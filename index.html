react项目中使用react-redux基本步骤

1.引入redux

yarn add redux

2.引入react-redux

yarn add react-redux

3.创建store文件夹，store中创建index.js、reducer.js，完成store基本结构

index.js文件中

import { createStore } from 'redux';
import reducer from './reducer';

const store = createStore(reducer);

export default store;

reducer.js文件中

const defaultState = {
    // 此处存储的是初始化的store数据，即原来存在组件state中的数据
    focused: false    //假设现在有一个focused的数据
};

export default (state=defaultState, action) => {
    if (action.type === 'search_focus') {
        return {
            focused: true
        }
    }
    if (action.type === 'search_blur') {
        return {
            focused: false
        }
    }
    return state;
}

4.根组件App.js中引入store和Provider并包裹子组件，则任意子组件都可使用store

import { Provider } from 'react-redux';
import store from './store'
class App extends Component {
      render() {
        return (
          <Provider store={store}>
            <内部组件 />
          </Provider>
        );
      }
}

5.在子组件中使用connect连接组件和store

import { connect } from 'react-redux';

// 原来组件中使用this.state.focused的地方则可替换为this.props.focused
// 原来组件中调用的方法this.handleInputFocus的地方则可替换为this.props.handleInputFocus

const mapStateTopProps = (state) => {
    return {
        focused: state.focused    // 将store中的focused映射到组件的props中去
    }
}
const mapDispatchToProps = (dispatch) => {
    return {
         handleInputFocus () {
            // 创建action
            const action = {
                type: 'search_focus'
            };
            // 将action派发给store
            dispatch(action);
        },
        handleInputBlur () {
            const action = {
                type: 'search_blur'
            };
            dispatch(action);
        }
    }
}
export default connect(mapStateToProps, mapDispatchToProps)(子组件);

hexo搭建博客托管在github上基本步骤

1.新建GitHub仓库，仓库名为：username.github.io

2.安装Git、Node.js

3.安装hexo脚手架

npm install -g hexo-cli -g

4.创建博客

hexo init username.github.io

5.更改配置

打开配置文件username.github.io/_config.yml

title: dimsky 的 9 维空间 //你博客的名字 
author: dimsky //你的名字
language: zh-Hans //语言 中文
theme: landscape //默认主题
deploy:
    type: git //使用Git 发布
    repo: git@github.com:username/username.github.io.git // 刚创建的Github仓库地址，需要添加ssh key到github账号
    branch: master

6.新建文章

hexo new article_name

打开文件夹username.github.io/source/_posts，打开刚才的新建的文件，编写即可。

7.本地测试

hexo server

然后打开localhost:4000端口即可浏览

8.安装hexo-deployer-git自动化部署

npm install hexo-deployer-git --save

9.发布到github

hexo clean && hexo g && hexo d

若第8步执行完后执行第9步报如下错误

ERROR Local hexo not found in 本地磁盘
ERROR Try running: 'npm install hexo --save'

则运行如下命令即可

npm install

浏览器打开https://username.github.io即可访问博客

10.每次写完文章后执行命令9即可同步到github

11.若需要更改主题，将主题clone到themes目录下，再修改配置文件中的theme为clone下的文件名，并theme文档修改配置即可

参考链接：
https://www.jianshu.com/p/4eaddcbe4d12
